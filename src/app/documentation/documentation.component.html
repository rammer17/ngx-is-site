Introduction

NGX-IS Goals

NGX-IS aims to provide a seamless user experience by creating intelligent and adaptive UI elements that automatically conform to the unique aesthetic of your application. It leverages native browser APIs and form elements, while also incorporating powerful components, empowering developers to create user-centric web applications with ease.
Created by the community, for the community.

Benefits

  • Faster Development: Leverage pre-built components to accelerate development and reduce time-to-market.
  • Consistent Design: Ensure consistency across your applications with unified styles and reusable elements.
  • Enhanced User Experience: Deliver intuitive and engaging interfaces that keep users coming back.
  • Flexibility and Customization: Tailor components to fit your unique design requirements with ease.

Powerful table component for displaying data in tabular format.

Usage

Import the data table component. To function properly an array of data must be provided to the component. Optionally a list of actions and filters can be passed too. The component handles sorting, filtering, toggling columns, paginating all on it's own, no need for further configuring 
                      
                          import { DataTableComponent } from 'ngx-is';
