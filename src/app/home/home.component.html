The Complete Component Library for Angular

NGX-IS allows you to create adaptive and accessible interfaces for web apps of any shape or size.

Designed for Angular applications

NGX-IS integrates directly with Angular and it's best features, including Component-Based Architecture, Dependency Injection, Directives, TypeScript Integration, and much more.

Enhanced with SCSS.

SCSS offers enhanced CSS functionality with features like variables and nesting, making stylesheets more maintainable and organized. Its support for mixins, inheritance, and mathematical operations improves code reusability and promotes a modular approach to styling, resulting in cleaner and more efficient CSS code.
